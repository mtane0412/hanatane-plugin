# 7つの本質的な質問: Michael Bungay Stanier式コーチング

## 概要

Michael Bungay Stanierは、『The Coaching Habit』(邦題『コーチング・ハビット』)で、効果的なコーチングのための7つの質問を体系化しました。これらの質問は、シンプルでありながら強力で、本質に迫り、行動を促す力があります。

## 核心的な原則

### コーチング・ハビットの3つの柱

#### 1. Stay Curious a Little Bit Longer
(もう少しだけ、好奇心を持ち続けよう)

**原則**:
- すぐに答えを与えない
- 質問を続ける
- 相手に考えさせる

**理由**:
- 最初の答えは、大抵最良の答えではない
- 深掘りすることで、真の洞察が現れる
- 相手の自己発見を促す

#### 2. Focus on the Real Problem
(本当の問題に焦点を当てる)

**原則**:
- 表面的な問題で満足しない
- 「本当の課題は何か?」と問い続ける
- 症状ではなく、根本原因を探る

**理由**:
- 最初に話題になる問題は、多くの場合、本当の問題ではない
- 本質的な問題を解決しなければ、また同じ問題が起こる
- 深い問題に取り組むことで、大きな変化が生まれる

#### 3. Say Less, Ask More
(話すより、質問する)

**原則**:
- アドバイスを控える
- 質問で導く
- 相手に話させる

**理由**:
- アドバイスは大抵無視される
- 自分で見つけた答えは、実行される
- 質問は、相手の思考を促す

## 7つの本質的な質問

### 1. The Kickstart Question (きっかけの質問)

#### 質問

**「What's on your mind?」**
(何を考えていますか?)

#### 目的

- 対話を始める
- 本当に重要なことに焦点を当てる
- 表面的な話題を超える

#### なぜ効果的か

**問題**:
- 「調子はどう?」は、社交辞令的な返答を引き出す
- 「何か助けが必要?」は、Noと言いやすい
- 話題が散漫になりやすい

**Kickstart Questionの効果**:
- オープンで、正直な返答を促す
- 相手が本当に考えていることに焦点を当てる
- 本質的な対話への入り口となる

#### 使い方

**基本形**:
```
「What's on your mind?」
```

**バリエーション**:
```
「何が最も重要ですか?」
「今、頭の中で最大の位置を占めているのは何ですか?」
「何から話しましょうか?」
```

**執筆への応用**:
```
「この記事で最も伝えたいことは何ですか?」
「読者に最も知ってほしいことは何ですか?」
「何があなたをこのテーマに引きつけましたか?」
```

#### 3Pモデル

Kickstart Questionの返答は、通常3つのカテゴリのいずれかに分類されます:

**Project (プロジェクト)**:
- やるべきタスクや仕事
- 「このプロジェクトを終わらせないと」

**People (人間関係)**:
- 他者との関係
- 「Xとの関係がうまくいっていない」

**Pattern (パターン)**:
- 繰り返されるパターンや習慣
- 「いつも同じ問題に直面する」

**深掘りの方向**:
```
「それはProject、People、Patternのどれに関することですか?」
```

### 2. The AWE Question (畏敬の質問)

#### 質問

**「And what else?」**
(他には?)

#### 目的

- 深掘りする
- 最初の答えを超える
- より良いアイデアを引き出す

#### なぜ効果的か

**問題**:
- 最初の答えは、大抵最良の答えではない
- 相手は、最も言いやすいことを最初に言う
- 本当に重要なことは、後から出てくる

**AWE Questionの効果**:
- シンプルで、安全
- 何度でも使える
- 真の洞察を引き出す

#### 使い方

**基本形**:
```
ユーザー: 「Aについて書きたいです」
あなた: 「他には?」
ユーザー: 「Bもあります」
あなた: 「他には?」
ユーザー: 「Cも重要です」
あなた: 「他には?」
ユーザー: 「実は、Dが最も重要かもしれません」
```

**推奨回数**: 3-5回

**注意**:
- 6回以上は、尋問のように感じられる
- 相手が「それだけです」と言ったら、止める
- 無理に引き出さない

**バリエーション**:
```
「他に何かありますか?」
「それ以外には?」
「まだありますか?」
「他の側面は?」
```

**執筆への応用**:
```
ユーザー: 「技術的な解説を書きます」
あなた: 「他には?」
ユーザー: 「実際の使用例も」
あなた: 「他には?」
ユーザー: 「落とし穴も説明したい」
あなた: 「他には?」
ユーザー: 「実は、なぜこの技術を選んだかの背景も重要です」
```

#### AWE Questionの魔法

> 「The best question is often: And what else?」
> (最良の質問は、しばしば「他には?」だ)

**なぜ魔法なのか**:
1. **シンプル**: 覚えやすく、使いやすい
2. **安全**: 批判的に聞こえない
3. **効果的**: 深い洞察を引き出す
4. **汎用的**: どんな状況でも使える
5. **習慣化しやすい**: 自然に使えるようになる

### 3. The Focus Question (焦点の質問)

#### 質問

**「What's the real challenge here for you?」**
(あなたにとって、本当の課題は何ですか?)

#### 目的

- 本質的な問題を特定する
- 表面的な症状ではなく、根本原因に迫る
- 個人的な関わりを明確にする

#### なぜ効果的か

**問題**:
- 最初に話題になる問題は、多くの場合、本当の問題ではない
- 人は、解決しやすい問題を先に話す傾向がある
- 本質的な問題は、隠れていることが多い

**Focus Questionの効果**:
- 「real challenge」が核心に迫る
- 「for you」が個人的な関わりを明確にする
- 一般論ではなく、個人的な課題に焦点を当てる

#### 使い方

**基本形**:
```
「What's the real challenge here for you?」
```

**バリエーション**:
```
「あなたにとって、本当の難しさは何ですか?」
「この中で、最も重要な課題は何ですか?」
「何があなたを阻んでいますか?」
「本質的な問題は何だと思いますか?」
```

**執筆への応用**:
```
「この記事を書く上で、あなたにとって本当の課題は何ですか?」
「読者にとって、本当の難しさは何ですか?」
「なぜこのテーマが重要だと感じるのですか?」
```

#### "for you"の重要性

**"for you"がない場合**:
```
「本当の課題は何ですか?」
→ 一般的な、抽象的な回答を引き出す
```

**"for you"がある場合**:
```
「あなたにとって、本当の課題は何ですか?」
→ 個人的な、具体的な回答を引き出す
```

**例**:
```
❌ 「課題は何ですか?」
   回答: 「業界全体の問題として、XXがあります」

✅ 「あなたにとって、本当の課題は何ですか?」
   回答: 「私は、XXについて確信が持てないんです」
```

### 4. The Foundation Question (基盤の質問)

#### 質問

**「What do you want?」**
(何が欲しいのですか?)

#### 目的

- 明確なゴールを設定する
- 曖昧さを排除する
- 行動を促す

#### なぜ効果的か

**問題**:
- 人は、何が欲しいのか明確にしていないことが多い
- 問題を語るが、ゴールを語らない
- ゴールがないと、行動につながらない

**Foundation Questionの効果**:
- 具体的なゴールを引き出す
- 前向きな焦点を作る
- 行動の基盤を築く

#### 使い方

**基本形**:
```
「What do you want?」
```

**バリエーション**:
```
「何を達成したいですか?」
「理想的な結果は何ですか?」
「これが終わったとき、何が起こっていますか?」
「成功とは、どんな状態ですか?」
```

**執筆への応用**:
```
「この記事を通して、何を達成したいですか?」
「読者に何を持ち帰ってほしいですか?」
「記事を読んだ後、読者にどうなってほしいですか?」
「この記事が成功したと言えるのは、どんな時ですか?」
```

#### ゴール設定のフレームワーク

**良いゴールの条件**:
1. **具体的** (Specific): 明確で測定可能
2. **肯定的** (Positive): 「〜しない」ではなく「〜する」
3. **現実的** (Realistic): 達成可能
4. **行動可能** (Actionable): 次のステップが明確

**例**:
```
❌ 悪いゴール: 「良い記事を書きたい」
   (曖昧、測定不可能)

✅ 良いゴール: 「読者が実際に試せる、3つの具体的な
   テクニックを伝える記事を書きたい」
   (具体的、測定可能、行動可能)
```

### 5. The Lazy Question (怠惰な質問)

#### 質問

**「How can I help?」**
(どう手伝えますか?)

#### 目的

- 相手に主体性を持たせる
- 的確なサポートを提供する
- 過剰な支援を避ける

#### なぜ効果的か

**問題**:
- 過剰なアドバイスは、相手の自律性を奪う
- 的外れなサポートは、時間の無駄
- 相手が本当に必要としていることを聞かない

**Lazy Questionの効果**:
- 相手に、必要なサポートを定義させる
- 的確な支援を提供できる
- 相手の主体性を尊重する

#### 使い方

**基本形**:
```
「How can I help?」
```

**バリエーション**:
```
「どの部分で最も助けが必要ですか?」
「何があれば、これを達成できますか?」
「どんなサポートが役立ちますか?」
「私に何ができますか?」
```

**執筆への応用**:
```
「記事を書く上で、どの部分で最も助けが必要ですか?」
「何があれば、この記事をより良くできますか?」
「どんなフィードバックが役立ちますか?」
「次のステップで、どう手伝えますか?」
```

#### 「怠惰」の意味

**なぜ"Lazy"なのか**:
- 自分で答えを考えなくて良い(相手に聞く)
- 効率的(的確なサポートのみ提供)
- エネルギーを節約(無駄な努力をしない)

**"Lazy Question"の哲学**:
> 「The lazy coach asks the client to do the work, rather than doing it for them」
> (怠惰なコーチは、クライアントに仕事をさせる。代わりにやってあげるのではなく)

### 6. The Strategic Question (戦略的質問)

#### 質問

**「If you're saying Yes to this, what are you saying No to?」**
(これにYesと言うなら、何にNoと言っていますか?)

#### 目的

- トレードオフを明確にする
- 優先順位をつける
- 焦点を絞る

#### なぜ効果的か

**問題**:
- 人は、全てを含めたがる
- Yesと言うことのコストを考えない
- 焦点が散漫になる

**Strategic Questionの効果**:
- トレードオフを顕在化させる
- 選択を意識的にする
- 焦点を絞る助けとなる

#### 使い方

**基本形**:
```
「If you're saying Yes to this, what are you saying No to?」
```

**バリエーション**:
```
「これを含めるなら、何を省きますか?」
「これに焦点を当てるなら、何を犠牲にしますか?」
「何を優先し、何を後回しにしますか?」
「限られた時間で、何を選び、何を捨てますか?」
```

**執筆への応用**:
```
「この内容を含めるなら、何を省きますか?」
「読者の注意は限られています。何に焦点を当てますか?」
「このセクションを詳しく書くなら、どのセクションを短くしますか?」
「必須の内容は何で、オプションの内容は何ですか?」
```

#### 戦略とは選択である

**ストラテジーの本質**:
> 「Strategy is about saying No」
> (戦略とは、Noと言うことだ)

**選択のマトリクス**:
```
           重要度
           高 | 低
        -------+-------
時間  多 | A  | B
必要  少 | C  | D

A: やる(最優先)
B: 省く(時間の無駄)
C: 効率化する
D: 絶対に省く
```

### 7. The Learning Question (学びの質問)

#### 質問

**「What was most useful for you?」**
(何が最も役に立ちましたか?)

#### 目的

- 学びを定着させる
- 次につなげる
- 成長を促す

#### なぜ効果的か

**問題**:
- 対話の内容は、忘れられやすい
- 学びが定着しない
- 次の行動につながらない

**Learning Questionの効果**:
- 学びを言語化させる
- 記憶に定着させる
- 次の行動への橋渡しとなる

#### 使い方

**基本形**:
```
「What was most useful for you?」
```

**バリエーション**:
```
「このセッションで最も価値があったことは何ですか?」
「どの洞察が最も重要でしたか?」
「何を覚えておきたいですか?」
「次に何をしますか?」
```

**執筆への応用**:
```
「このインタビューで最も価値があったことは何ですか?」
「どのアイデアが最も重要でしたか?」
「次のステップは何ですか?」
「いつから書き始めますか?」
```

#### 学習の定着

**学習のサイクル**:
```
1. 経験する(対話)
   ↓
2. 振り返る(Learning Question)
   ↓
3. 言語化する(回答)
   ↓
4. 行動する(次のステップ)
```

**Learning Questionの効果**:
- 経験を反芻させる
- 重要な部分を特定させる
- コミットメントを引き出す

## 7つの質問の順序と組み合わせ

### 基本的な流れ

```
1. Kickstart Question
   「何を考えていますか?」
   → 対話の入り口

2. AWE Question (3-5回)
   「他には?」
   → 深掘りと探求

3. Focus Question
   「本当の課題は何ですか?」
   → 本質の特定

4. Foundation Question
   「何が欲しいのですか?」
   → ゴールの設定

5. Lazy Question
   「どう手伝えますか?」
   → サポートの特定

6. Strategic Question
   「何にYes、何にNoですか?」
   → 優先順位の決定

7. Learning Question
   「何が最も役立ちましたか?」
   → 学びの定着
```

### 柔軟な使い方

**必ずしも全ての質問を使う必要はありません**:
- 状況に応じて選択する
- 重複する質問は省く
- 最も効果的な質問に焦点を当てる

**最小限のセット**:
```
1. Kickstart Question (開始)
2. AWE Question (深掘り)
3. Focus Question (本質)
4. Learning Question (定着)
```

## 実践のガイドライン

### DO: すべきこと

1. **順番を守る**: 特にKickstart → AWE → Focusの流れは効果的
2. **AWE Questionを多用する**: 3-5回は使う
3. **"for you"を忘れない**: 個人的な関わりを明確にする
4. **沈黙を許容する**: 深い質問には時間がかかる
5. **シンプルに**: 質問は短く、明確に
6. **好奇心を持つ**: 本当に知りたいという姿勢で
7. **Learning Questionで終わる**: 学びを定着させる

### DON'T: すべきでないこと

1. **質問を複雑にしない**: シンプルが最良
2. **質問を詰め込まない**: 一度に1つの質問
3. **答えを誘導しない**: オープンな質問を保つ
4. **すぐにアドバイスしない**: Stay curious
5. **表面的な答えで満足しない**: AWE Questionで深掘り
6. **"for you"を省かない**: 個人的な関わりが鍵
7. **Learning Questionを忘れない**: 必ず最後に使う

## まとめ

Michael Bungay Stanierの「7つの本質的な質問」の本質:

**核心的な原則**:
1. Stay curious a little bit longer(もう少し好奇心を)
2. Focus on the real problem(本当の問題に焦点を)
3. Say less, ask more(話すより質問を)

**7つの質問**:
1. Kickstart: 何を考えていますか?
2. AWE: 他には?(魔法の質問)
3. Focus: 本当の課題は何ですか?
4. Foundation: 何が欲しいのですか?
5. Lazy: どう手伝えますか?
6. Strategic: 何にYes、何にNoですか?
7. Learning: 何が最も役立ちましたか?

**実践の鍵**:
- シンプルに保つ
- 順番を守る
- AWE Questionを多用する
- "for you"を忘れない
- Learning Questionで終わる

この手法を使って、本質に迫り、行動を促すコーチングを実践してください。
