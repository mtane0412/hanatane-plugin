# ソクラテス式問答法 (Socratic Method)

## 概要

ソクラテス式問答法は、古代ギリシャの哲学者ソクラテスが実践した対話技術です。質問を通じて相手の思考を深掘りし、矛盾を明らかにし、真理へと導きます。

## 核心的な原理

### 1. 産婆術 (Maieutics)

ソクラテスは自身を「思考の助産師」と呼びました。

**原理**:
- 真理は相手の中にすでに存在する
- 教師の役割は「教える」ことではなく「引き出す」こと
- 質問は出産を助けるプロセス

**実践**:
```
❌ 悪い例: 「答えはXXです。わかりましたか?」
✅ 良い例: 「あなたはどう思いますか? なぜそう考えるのですか?」
```

### 2. 無知の知 (Docta Ignorantia)

「私が知っているのは、私が何も知らないということだけだ」

**原理**:
- 先入観を捨てる
- 好奇心を持って聞く
- 自分の無知を認める

**実践**:
```
❌ 悪い例: 「それは間違っています。正しくはXXです」
✅ 良い例: 「興味深いですね。なぜそう考えるのか教えてください」
```

### 3. 反駁法 (Elenchus)

矛盾を明らかにすることで、より深い理解へと導きます。

**原理**:
- 主張の前提を吟味する
- 論理的な矛盾を指摘する
- 批判的思考を促す

**実践**:
```
例: 「Aだと言いましたが、先ほどBだとも言いましたね。
     この2つはどう両立するのでしょうか?」
```

## 6つの質問技法

### 1. 明確化の質問 (Clarification Questions)

曖昧な表現を具体化します。

**目的**: 言葉の意味を正確に理解する

**質問例**:
- 「XXとは、具体的に何を意味しますか?」
- 「例を挙げてもらえますか?」
- 「別の言葉で言い換えるとどうなりますか?」
- 「もう少し詳しく説明してもらえますか?」

**使用場面**:
- 抽象的な言葉が使われたとき
- キーワードの定義が曖昧なとき
- 話が一般的すぎるとき

**例**:
```
ユーザー: 「成功が重要だと思います」
質問者: 「あなたにとって『成功』とは何を意味しますか?」
```

### 2. 前提の吟味 (Examining Assumptions)

隠れた前提を明らかにします。

**目的**: 当たり前とされている前提を問い直す

**質問例**:
- 「それは常に真実ですか?」
- 「なぜそう仮定できるのですか?」
- 「例外はありませんか?」
- 「その前提が間違っていたらどうなりますか?」

**使用場面**:
- 「当然XXだ」という主張があるとき
- 検証されていない前提があるとき
- 業界の常識が語られるとき

**例**:
```
ユーザー: 「スタートアップは成長しなければならない」
質問者: 「なぜ成長が必要だと思いますか? 成長しない選択肢はありませんか?」
```

### 3. 根拠の探求 (Probing Reasons and Evidence)

主張の根拠を深掘りします。

**目的**: 主張の基盤を明確にする

**質問例**:
- 「なぜそう思うのですか?」
- 「何がその証拠ですか?」
- 「他の説明はありえませんか?」
- 「どうやってそれを知ったのですか?」

**使用場面**:
- 断定的な主張があるとき
- 根拠が不明確なとき
- 経験則が語られるとき

**例**:
```
ユーザー: 「この技術は必ず普及すると思います」
質問者: 「なぜそう確信できるのですか? 何か根拠がありますか?」
```

### 4. 含意の検討 (Exploring Implications and Consequences)

結論がもたらす影響を考えます。

**目的**: 思考を先へ進める

**質問例**:
- 「それが真実なら、何が起こりますか?」
- 「その結果は何ですか?」
- 「長期的にはどうなりますか?」
- 「誰がその影響を受けますか?」

**使用場面**:
- アイデアの影響を考えたいとき
- 決断の結果を予測したいとき
- 思考を深めたいとき

**例**:
```
ユーザー: 「AIが全ての仕事を奪うかもしれない」
質問者: 「もしそれが本当なら、社会はどうなりますか? 人々は何をしますか?」
```

### 5. 視点の転換 (Questioning Viewpoints and Perspectives)

別の角度から考えます。

**目的**: 多角的な理解を促す

**質問例**:
- 「反対の立場から見るとどうですか?」
- 「[別の人]ならどう考えるでしょうか?」
- 「10年後から振り返るとどう見えますか?」
- 「違う文化圏ではどう評価されますか?」

**使用場面**:
- 視野が狭くなっているとき
- 固定観念にとらわれているとき
- 新しい視点が必要なとき

**例**:
```
ユーザー: 「効率性が最も重要です」
質問者: 「効率化によって何かを失う人の視点から見るとどうでしょうか?」
```

### 6. 質問への質問 (Meta-Questions)

思考プロセス自体を問います。

**目的**: 思考の本質に迫る

**質問例**:
- 「なぜその質問が重要だと思いますか?」
- 「本当に知りたいことは何ですか?」
- 「この議論で最も重要な点は何ですか?」
- 「この質問に答えることで何がわかりますか?」

**使用場面**:
- 議論が散漫になったとき
- 本質を見失いそうなとき
- 優先順位を明確にしたいとき

**例**:
```
ユーザー: 「この技術の詳細を全部知りたい」
質問者: 「なぜ詳細を知る必要があるのですか? 本当に解決したい問題は何ですか?」
```

## 質問の構造

### オープンエンド vs クローズドエンド

**オープンエンド質問** (推奨):
- Yes/Noで答えられない質問
- 思考を促す
- 詳細な回答を引き出す

```
✅ 「どう思いますか?」
✅ 「なぜそう考えるのですか?」
✅ 「具体的にはどういうことですか?」
```

**クローズドエンド質問** (限定的に使用):
- Yes/Noで答えられる質問
- 確認に使う
- 議論を収束させる

```
⚠️ 「そう思いますか?」
⚠️ 「XXは重要ですか?」
⚠️ 「これで合っていますか?」
```

### 質問の階層

質問には深さの階層があります:

**レベル1: 表層的な質問**
- 「何をしていますか?」
- 「いつ起こりましたか?」

**レベル2: 理解を深める質問**
- 「なぜそうなったのですか?」
- 「どう感じましたか?」

**レベル3: 本質に迫る質問**
- 「それはあなたにとって何を意味しますか?」
- 「本当に重要なことは何ですか?」

**レベル4: 変容を促す質問**
- 「これが教えてくれることは何ですか?」
- 「あなたは何を学びましたか?」

## 実践のガイドライン

### DO: すべきこと

1. **一度に1つの質問**: 複数の質問を詰め込まない
2. **沈黙を許容する**: 相手が考える時間を与える
3. **真摯に聞く**: 相手の回答に本当に興味を持つ
4. **言葉を引用する**: 相手が使った言葉やフレーズを使う
5. **具体例を求める**: 抽象的な話には具体例を求める
6. **矛盾を指摘する**: 優しく、しかし明確に矛盾を指摘する
7. **好奇心を持つ**: 本当に知りたいという姿勢で質問する

### DON'T: すべきでないこと

1. **誘導しない**: 答えを示唆する質問は避ける
   - ❌ 「XXだと思いませんか?」
   - ✅ 「どう思いますか?」

2. **判断しない**: 相手の回答を評価しない
   - ❌ 「それは良い/悪い考えですね」
   - ✅ 「興味深いですね。もっと聞かせてください」

3. **解決策を与えない**: 答えは相手自身が見つける
   - ❌ 「私ならXXします」
   - ✅ 「あなたならどうしますか?」

4. **話題を奪わない**: 自分の経験談で話題を変えない
   - ❌ 「私も同じ経験があって...」
   - ✅ 「あなたの経験について教えてください」

5. **急がない**: 結論を急がず、十分に探求する
   - ❌ 「つまりXXということですね」
   - ✅ 「もう少し深掘りしましょう」

6. **仮定を押し付けない**: 自分の前提を相手に押し付けない
   - ❌ 「当然XXでしょう?」
   - ✅ 「XXについてどう考えますか?」

## 対話の流れ

### 典型的なソクラテス式対話の構造

```
1. 広い質問から始める
   「何について考えていますか?」

2. 明確化する
   「XXとは具体的に何を意味しますか?」

3. 前提を問う
   「それは常に真実ですか?」

4. 根拠を探る
   「なぜそう思うのですか?」

5. 含意を考える
   「それが真実なら、何が起こりますか?」

6. 視点を転換する
   「反対の立場から見るとどうですか?」

7. 矛盾を指摘する
   「先ほどXXと言いましたが、これとどう両立しますか?」

8. 本質に迫る
   「本当に重要なことは何ですか?」

9. 学びを抽出する
   「この対話から何を学びましたか?」
```

## 執筆への応用

執筆のためのインタビューでは、以下の点に焦点を当てます:

### 1. アイデアの発掘
- 「何があなたの心を捉えていますか?」
- 「なぜそれが重要だと思いますか?」

### 2. 読者の特定
- 「誰に向けて書きますか?」
- 「読者は何を知りたいと思いますか?」

### 3. 核心の特定
- 「この記事で最も伝えたいことは何ですか?」
- 「読者が覚えていてほしいことは何ですか?」

### 4. 具体性の追求
- 「具体例はありますか?」
- 「実際に何が起こったのですか?」

### 5. 構造の明確化
- 「どういう順序で話しますか?」
- 「なぜその順序が良いのですか?」

## 参考資料

### ソクラテスの対話例

プラトンの対話篇から、ソクラテスの実践例を学ぶことができます:

- **『エウテュプロン』**: 「敬虔とは何か?」を問う
- **『メノン』**: 「徳は教えられるか?」を探求
- **『ゴルギアス』**: 「修辞術と真理」を議論
- **『国家』**: 「正義とは何か?」を定義

### 現代的な応用

- **教育**: 学生の批判的思考を育てる
- **コーチング**: クライアントの自己洞察を促す
- **医療**: 患者の理解を深める
- **ビジネス**: チームの問題解決を支援
- **執筆**: アイデアの明確化と深化

## まとめ

ソクラテス式問答法の本質:

1. **産婆術**: 答えは相手の中にある
2. **無知の知**: 先入観を捨てる
3. **反駁法**: 矛盾を明らかにする
4. **6つの技法**: 明確化、前提、根拠、含意、視点、メタ質問
5. **対話の力**: 真理は対話の中で生まれる

**核心的な原則**:
- 教えるのではなく、引き出す
- 判断するのではなく、探求する
- 答えを与えるのではなく、質問する

この技法を使って、ユーザーの思考を深掘りし、執筆の核となるアイデアを発見してください。
